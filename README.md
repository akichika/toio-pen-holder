# toio Pen Holder

A 3D printable pen holder attachment designed for the toio™ cube robot, enabling creative drawing activities in educational workshops such as CoderDojo.

## Features

- **LEGO Compatible**: Built with standard LEGO dimensions (16mm width)
- **Two Variants**: 
  - Simple type: Easy to 3D print
    <img width="275" height="230" alt="スクリーンショット 2026-01-05 022242" src="https://github.com/user-attachments/assets/c09831af-51ec-4e13-a6c0-07744a16575f" />
  - Stud type: Can attach additional LEGO blocks
    <img width="275" height="230" alt="スクリーンショット 2026-01-05 022156" src="https://github.com/user-attachments/assets/1e223fc4-8db0-4b01-922f-a4b30d21018d" />
- **Multiple Pen Support**: Up to 4 pens can be attached simultaneously
- **Versatile**: Fits various pen sizes
- **Reusable**: Designed for repeated workshop use

## Usage

1. Attach the pen holder to your toio cube
2. Insert a pen into the holder
3. Secure with a rubber band to prevent dropping
4. Create unique geometric drawings with toio's movements!

## Recommended Pens

- MUJI 10-Color Mini Color Pen Set
- Daiso 36-Color Pen Set (cut in half for workshops)

## Important Notes

⚠️ **Do NOT draw directly on**:
- toio identification cards
- toio dedicated mats (unless using erasable pens like Frixion)

⚠️ **Pen Limitations**:
- Designed for shorter/lighter pens
- For longer pens, extend the holder vertically
