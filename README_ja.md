# toio ペンホルダー

ロボットトイ「toio™」用の3Dプリント可能なペンホルダーです。CoderDojoなどのプログラミングワークショップなどでお絵描き用として活用いただけます。

## 特徴

- **レゴ互換**: レゴブロックと同じ基準寸法（16mm幅）で設計
- **2つのバリエーション**: 
  - シンプルタイプ: 3Dプリントしやすいミニマル設計
    <img width="275" height="230" alt="スクリーンショット 2026-01-05 022242" src="https://github.com/user-attachments/assets/c09831af-51ec-4e13-a6c0-07744a16575f" />
  - ポッチ付きタイプ: レゴブロックを追加で取り付け可能
    <img width="275" height="230" alt="スクリーンショット 2026-01-05 022156" src="https://github.com/user-attachments/assets/1e223fc4-8db0-4b01-922f-a4b30d21018d" />
- **複数ペン対応**: 最大4本まで同時に取り付け可能
- **汎用性**: 様々な太さのペンに対応
- **再利用可能**: ワークショップでの繰り返し使用を想定した設計

## 使い方

1. ペンホルダーをtoio™コア キューブに取り付ける
2. ペンをホルダーに差し込む
3. 輪ゴムで固定して落下を防止する
4. toioの動きで幾何学的な絵を描いてみよう！

## 重要な注意事項

⚠️ **以下には直接描かないでください**:
- トイオ・プレイグラウンドのカード（水生ペンで描いた場合、ふき取ったり洗うことである程度きれいになります）
- toio用簡易マット・開発用マット・その他マット類（フリクションペンなどの消せるペンを使う場合を除く）

⚠️ **ペンの制限**:
- 短め・軽めのペン向けに設計されています
- 長いペンを使う場合は、ホルダーを上に延長する必要があります

## ライセンス

CC-0（パブリックドメイン）

自由に使用、改変、共有してください！
